## 使用说明

- 安装 VSCode，并为 VSCode 安装 live server 插件，可[点击参考文档](https://sillytavern-stage-girls-dog.readthedocs.io/工具经验/_vscode/index.html)
- 用 VSCode 打开一个文件夹，在其中新建一个 test.css 文件
- 在 VSCode 文件列表右键 test.css 文件，点击 "复制相对路径"
- 在 VSCode 右下角点击 "Go Live" 来将文件夹映射到链接上，然后 "Go Live" 将会变成一串数字
- 启用 "实时修改CSS" 脚本
- 编辑 "实时修改CSS" 脚本，在`链接`中填入 `http://localhost:那一串数字/你复制得到的相对路径`
