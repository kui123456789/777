# 深度条目排斥器

- **作者:** 青空莉想做舞台少女的狗
- **版本:** 2025/11/13
- **源文件:** [点此跳转](https://gitgud.io/StageDog/tavern_resource/-/tree/main/src)

## 功能

在提示词实际发送时,

- 将 D10 以下的深度条目按序移动到 D0
- 将 D10 以上的深度条目按序移动到 D9999

## 原因

按照一些预设作者和角色卡作者的说法, gemini 和 claude 不同, 不必将条目插入聊天记录中, 插入其中反而会干扰聊天记录的连续性, 重要条目应该都插入到 D0.

但我喜欢混着 AI 玩: 偶尔用 gpt、偶尔用 claude、偶尔用 gemini……则显然让世界书条目继续使用 D4 等位置, 而用一个脚本来把这些条目在需要时移动到 D0 进行发送更好.

此外, 我测试发现 "D4 具体格式 + D0 输出强调" 在 gemini 上效果依旧很好, 而难以与 "D0 输出强调" 的效果进行对比——因此干脆不对比, 就按世界书条目放 D4 等位置写, 想放 D0 的自己开脚本😤
