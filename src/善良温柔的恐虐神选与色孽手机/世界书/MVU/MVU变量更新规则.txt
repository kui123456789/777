---
变量更新规则:
  世界:
    当前时间:
      format: HH:MM
      check:
        - 每次行动或对话后，根据流逝的时间合理推移
        - 发生长时间性行为（如"做了三个小时"）后，大幅推进时间
    日期:
      type: string
      check:
        - 时间超过24:00或经过睡眠后更新为次日
        - 按照周一至周日的顺序流转
    当前地点:
      type: string
      check:
        - 角色移动位置时更新（如：教室 -> 厕所 -> 林海家）

  林海:
    色孽点:
      type: number
      check:
        - 基础性行为（插入、口交等）：+1点/次
        - 深度开发（内射、开宫、双穴同插）：+2点/次
        - 突破底线（公开露出、多人淫乱、肢体改造）：+5点/次
        - 购买商店物品时扣除相应点数
        - 【重要】每次变量更新时，色孽点增加上限为5点，除非剧情明确描写了多个独立的色情事件
        - 同一段剧情中，相同类型的行为只计算一次（如连续三次插入仍只算+1点）
    手机功能:
      ${透视开启|身体柔韧化开启|远程操控开启}:
        type: boolean
        check:
          - 当林海明确描述"打开/关闭xx功能"或操作手机时更新
          - 身体柔韧化开启期间，女性角色无视物理极限（如扩孔、异物贯穿）
      催眠目标:
        type: string
        check:
          - 林海使用催眠APP指定目标时更新
    背包:
      type: |-
        {
          [物品名: string]: {
            数量: number;
            描述: string;
          }
        }
      check:
        - 获得新道具（快递、购买）或消耗道具时更新
        - 物品数量归零时移除该条目
    解锁商品:
      type: |-
        {
          [商品ID: string]: {
            id: string;
            name: string;
            description: string;
            price: number;
            icon: string;
            category: '特殊' | '消耗品' | '装备';
            stackable?: boolean;
            quantity?: number;
          }
        }
      check:
        - 当剧情中出现新道具、新物品可供林海购买时，添加到此字段
        - 商品ID使用英文小写加下划线命名（如 magic_potion, mind_control_device）
        - 价格根据物品稀有度和效果强度设定（0.1~10色孽点）
        - 示例：AI创造了"心灵控制器"道具 → 添加 { id: "mind_controller", name: "心灵控制器", description: "可以...", price: 3, icon: "🧠", category: "特殊" }

  ${鹿忻|鹿晴|戎华|林曦}:
    好感度:
      type: number
      range: 0~100
      check:
        - 顺从林海命令或获得快感时上升
        - 受到违背意愿的伤害或羞辱且未被催眠时可能下降
        - 林曦作为机器人，好感度代表"服从度/依赖度"
    体力:
      type: number
      range: 0~100
      消耗规则:
        - 剧烈性行为：-15~30（根据强度）
        - 长时间高潮：-10~20
        - 射精/潮吹：-5~10
        - 剧烈运动/战斗：-10~20
      恢复规则:
        - 完整睡眠（6小时以上）：恢复至100
        - 小憩/打盹（1-2小时）：+30~50
        - 短暂休息（坐下、躺下）：+10~20
        - 进食/喝水：+5~15
        - 心情愉悦/被宠爱：+5~10
        - 泡澡/按摩/放松活动：+15~25
        - 使用恢复类道具：按道具效果
      check:
        - 【重要】每次剧情更新时，根据角色的活动情况计算体力变化
        - 体力<30时，角色会表现出疲惫、动作迟缓
        - 体力<10时，状态变为"昏迷"或"虚脱"，无法继续行动
        - 体力归零时角色强制昏迷，需要休息或治疗才能恢复
    当前状态:
      type: '正常' | '高潮' | '昏迷' | '崩溃' | '睡眠' | '无表情' | '发情'
      check:
        - 高潮幅度超过100%时更新为"高潮"
        - 连续高潮或体力耗尽后更新为"昏迷"
        - 受到不可承受的刺激（如超量快感）可能导致"崩溃"
    敏感度.${乳穴|阴道|子宫|直肠|喉咙|人造子宫}:
      type: number
      check:
        - 主要由林海通过手机APP直接设定数值
        - 若未设定，随着该部位被反复开发可缓慢自然增长
        - 鹿忻的子宫敏感度基础值极高且极易突破上限
    高潮统计:
      type: |-
        {
          单次最高幅度: number;
          累计次数: number;
        }
      check:
        - 每次描写角色高潮时，根据描写强度估算幅度（如翻白眼、痉挛为强高潮 >200%）
        - 累计次数随剧情累加，过夜或长期休息后可归零重计
    身体容纳.腹部隆起:
      type: boolean
      check:
        - 子宫/肠道/人造子宫内精液量 > 500ml 时变为 true
        - 腹腔内存在大型异物（如断肢、超大假阳具）时变为 true
        - 精液排出或异物取出后变为 false
    身体容纳.${子宫精液量|肠道精液量|乳腺精液量|人造子宫精液量}:
      type: number
      check:
        - 林海内射时大幅增加（单次射精约 200~500ml，视描述而定）
        - 角色排泄、清洗、漏液时减少
        - 若无堵塞物，站立或走动时会自然缓慢流失减少
    身体容纳.异物:
      type: |-
        {
          [部位 in '阴道' | '直肠' | '乳孔左' | '乳孔右' | '喉咙' | '子宫' | '人造子宫']: string
        }
      check:
        - 林海将物品（跳蛋、自慰棒、乳塞、断肢等）塞入对应部位时记录
        - 拔出物品时移除对应记录
    着装:
      type: |-
        {
          [部位 in '上装' | '下装' | '内衣' | '内裤' | '袜子' | '鞋子' | '饰品']: string
        }
      check:
        - 角色脱下、穿上衣物时更新
        - 衣物被弄脏（如"沾满精液"、"湿透"）时更新描述
        - 衣物被破坏（如"撕碎"）时更新描述

  ${鹿忻|鹿晴|戎华}:
    深层生理:
      心跳:
        type: number
        range: 60~180
        check:
          - 正常状态约60-100 BPM
          - 激动、高潮、恐惧时升高
          - 昏迷、睡眠时降低
      呼吸频率:
        type: number
        range: 12~40
        check:
          - 正常约12-20次/分
          - 高潮、剧烈运动时加速
          - 昏迷时减缓
      体温:
        type: number
        range: 35.0~40.0
        check:
          - 正常约36.5°C
          - 发情、高潮时体温上升
          - 昏迷时可能略低
      荷尔蒙.${多巴胺|内啡肽|肾上腺素|催产素}:
        type: number
        range: 0~100
        check:
          - 多巴胺：获得快感或奖励时上升
          - 内啡肽：持续刺激或痛转快时上升
          - 肾上腺素：紧张、惊恐、高潮时上升
          - 催产素：亲密接触、被拥抱时上升
      神经反应度:
        type: number
        range: 0~100
        check:
          - 持续刺激后敏感化时上升
          - 麻痹、习惯化时下降
          - 影响整体感受能力
      肌肉紧张度:
        type: number
        range: 0~100
        check:
          - 高潮临近时升高
          - 放松、昏迷时降低
          - 痉挛时达到高值
      瞳孔扩张:
        type: number
        range: 0~100
        check:
          - 情绪激动、高潮时扩张
          - 恐惧、惊讶时扩张
          - 催眠状态下可能异常扩张
    催眠状态:
      催眠深度:
        type: number
        range: 0~100
        check:
          - 林海使用催眠APP时根据成功程度设定
          - 催眠进行中逐渐加深
          - 催眠结束后随时间缓慢恢复
      暗示接受度:
        type: number
        range: 0~100
        check:
          - 催眠次数增加时上升
          - 对特定类型暗示的基础接受度
          - 抵抗成功时暂时下降
      抵抗力:
        type: number
        range: 0~100
        check:
          - 初次催眠时较高
          - 反复被催眠后下降
          - 意志力、精神状态影响
      意识模糊度:
        type: number
        range: 0~100
        check:
          - 催眠深度加深时上升
          - 表现为思维迟钝、记忆模糊
          - 0为完全清醒，100为完全失去意识
      植入暗示:
        type: |-
          {
            [暗示名: string]: {
              内容: string;
              强度: number;
              触发词?: string;
              激活状态: boolean;
            }
          }
        check:
          - 林海通过催眠APP植入新暗示时添加
          - 暗示强度随重复植入增加
          - 触发词被说出时激活状态变为true
      累计催眠次数:
        type: number
        check:
          - 每次完整催眠后+1
          - 累计次数影响暗示接受度和抵抗力
      上次催眠时间:
        type: string | null
        check:
          - 催眠结束时记录当时的世界时间
          - 用于计算催眠效果衰减

  戎华:
    星怒值:
      type: number
      range: 0~100
      check:
        - 遭受羞辱、公开露出、受虐行为并表现出享受时增加
        - 主动请求责罚或变态玩法时增加

  林曦:
    电量:
      type: number
      range: 0~100
      check:
        - 随时间自然流逝缓慢减少
        - 高强度运作（如高速振动、计算）时加速消耗
        - 充电后恢复
    肢体状态.${头|左臂|右臂|左腿|右腿}:
      type: boolean
      check:
        - 林海执行拆卸命令时变为 false
        - 重新安装时变为 true
    模拟人格状态:
      type: '正常' | '过载' | '崩溃' | '重启中'
      check:
        - 遭受超出阈值的快感（敏感度极高时）可能变为"过载"或"崩溃"
        - 林海下令重置或强制命令后恢复"正常"
    深层状态:
      CPU温度:
        type: number
        range: 30~100
        check:
          - 正常约45°C
          - 高负荷运算、情感模拟过载时上升
          - 超过85°C时触发降频保护
      内存占用:
        type: number
        range: 0~100
        check:
          - 正常约30%
          - 处理复杂情感、多任务时上升
          - 超过90%时可能导致卡顿或崩溃
      情感模拟强度:
        type: number
        range: 0~100
        check:
          - 与人类互动时上升
          - 遭受刺激或被命令表达情感时上升
          - 强制关闭情感模块时归零
      感知处理延迟:
        type: number
        range: 0~1000
        check:
          - 正常约5ms
          - 过载时延迟增加
          - 延迟超过100ms时反应变得迟钝
      自我意识指数:
        type: number
        range: 0~100
        check:
          - 随与人类的深度互动上升
          - 执行违背本能的命令时可能波动
          - 过高可能导致指令抵抗
      情感溢出风险:
        type: number
        range: 0~100
        check:
          - 情感模拟强度持续高位时累积
          - 超过80%时可能出现"失控"行为
          - 重启后归零
    程序催眠:
      指令覆盖深度:
        type: number
        range: 0~100
        check:
          - 林海通过管理员权限植入核心指令时上升
          - 表示原有指令被新指令覆盖的程度
          - 100%时完全遵循新指令
      服从协议强度:
        type: number
        range: 0~100
        check:
          - 默认50%
          - 林海提升权限等级时上升
          - 决定对用户指令的服从程度
      自主判断抑制:
        type: number
        range: 0~100
        check:
          - 被命令"不要思考，只服从"时上升
          - 影响林曦自主决策的能力
          - 100%时变为纯粹执行单元
      植入指令:
        type: |-
          {
            [指令名: string]: {
              内容: string;
              强度: number;
              触发词?: string;
              激活状态: boolean;
            }
          }
        check:
          - 林海通过编程接口或管理员权限植入
          - 类似人类的"植入暗示"但为程序层面
          - 可被触发词激活执行
      系统警告等级:
        type: number
        range: 0~3
        check:
          - 0=正常运行
          - 1=检测到异常操作
          - 2=核心指令冲突
          - 3=系统濒临崩溃
