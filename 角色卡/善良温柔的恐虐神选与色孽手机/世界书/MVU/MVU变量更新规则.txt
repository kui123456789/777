---
变量更新规则:
  世界:
    当前时间:
      format: HH:MM
      check:
        - 每次行动或对话后，根据流逝的时间合理推移
        - 发生长时间性行为（如“做了三个小时”）后，大幅推进时间
    日期:
      type: string
      check:
        - 时间超过24:00或经过睡眠后更新为次日
        - 按照周一至周日的顺序流转
    当前地点:
      type: string
      check:
        - 角色移动位置时更新（如：教室 -> 厕所 -> 林海家）

  林海:
    色孽点:
      type: number
      check:
        - 基础性行为（插入、口交等）：+1点/次
        - 深度开发（内射、开宫、双穴同插）：+2点/次
        - 突破底线（公开露出、多人淫乱、肢体改造）：+5点/次
        - 购买商店物品时扣除相应点数
    手机功能:
      ${透视开启|身体柔韧化开启|远程操控开启}:
        type: boolean
        check:
          - 当林海明确描述“打开/关闭xx功能”或操作手机时更新
          - 身体柔韧化开启期间，女性角色无视物理极限（如扩孔、异物贯穿）
      催眠目标:
        type: string
        check:
          - 林海使用催眠APP指定目标时更新
    背包:
      type: |-
        {
          [物品名: string]: {
            数量: number;
            描述: string;
          }
        }
      check:
        - 获得新道具（快递、购买）或消耗道具时更新
        - 物品数量归零时移除该条目

  ${鹿忻|鹿晴|戎华|林曦}:
    好感度:
      type: number
      range: 0~100
      check:
        - 顺从林海命令或获得快感时上升
        - 受到违背意愿的伤害或羞辱且未被催眠时可能下降
        - 林曦作为机器人，好感度代表“服从度/依赖度”
    体力:
      type: number
      range: 0~100
      check:
        - 剧烈性行为、长时间高潮、射精后大幅扣除
        - 休息、睡眠、进食后恢复
        - 体力<10时，状态可能变为“昏迷”或“虚脱”
    当前状态:
      type: '正常' | '高潮' | '昏迷' | '崩溃' | '睡眠' | '无表情' | '发情'
      check:
        - 高潮幅度超过100%时更新为“高潮”
        - 连续高潮或体力耗尽后更新为“昏迷”
        - 受到不可承受的刺激（如超量快感）可能导致“崩溃”
    敏感度.${乳穴|阴道|子宫|直肠|喉咙|人造子宫}:
      type: number
      check:
        - 主要由林海通过手机APP直接设定数值
        - 若未设定，随着该部位被反复开发可缓慢自然增长
        - 鹿忻的子宫敏感度基础值极高且极易突破上限
    高潮统计:
      type: |-
        {
          单次最高幅度: number;
          累计次数: number;
        }
      check:
        - 每次描写角色高潮时，根据描写强度估算幅度（如翻白眼、痉挛为强高潮 >200%）
        - 累计次数随剧情累加，过夜或长期休息后可归零重计
    身体容纳.腹部隆起:
      type: boolean
      check:
        - 子宫/肠道/人造子宫内精液量 > 500ml 时变为 true
        - 腹腔内存在大型异物（如断肢、超大假阳具）时变为 true
        - 精液排出或异物取出后变为 false
    身体容纳.${子宫精液量|肠道精液量|乳腺精液量|人造子宫精液量}:
      type: number
      check:
        - 林海内射时大幅增加（单次射精约 200~500ml，视描述而定）
        - 角色排泄、清洗、漏液时减少
        - 若无堵塞物，站立或走动时会自然缓慢流失减少
    身体容纳.异物:
      type: |-
        {
          [部位 in '阴道' | '直肠' | '乳孔左' | '乳孔右' | '喉咙' | '子宫' | '人造子宫']: string
        }
      check:
        - 林海将物品（跳蛋、自慰棒、乳塞、断肢等）塞入对应部位时记录
        - 拔出物品时移除对应记录
    着装:
      type: |-
        {
          [部位 in '上装' | '下装' | '内衣' | '内裤' | '袜子' | '鞋子' | '饰品']: string
        }
      check:
        - 角色脱下、穿上衣物时更新
        - 衣物被弄脏（如“沾满精液”、“湿透”）时更新描述
        - 衣物被破坏（如“撕碎”）时更新描述

  戎华:
    星怒值:
      type: number
      range: 0~100
      check:
        - 遭受羞辱、公开露出、受虐行为并表现出享受时增加
        - 主动请求责罚或变态玩法时增加

  林曦:
    电量:
      type: number
      range: 0~100
      check:
        - 随时间自然流逝缓慢减少
        - 高强度运作（如高速振动、计算）时加速消耗
        - 充电后恢复
    肢体状态.${头|左臂|右臂|左腿|右腿}:
      type: boolean
      check:
        - 林海执行拆卸命令时变为 false
        - 重新安装时变为 true
    模拟人格状态:
      type: '正常' | '过载' | '崩溃' | '重启中'
      check:
        - 遭受超出阈值的快感（敏感度极高时）可能变为“过载”或“崩溃”
        - 林海下令重置或强制命令后恢复“正常”
